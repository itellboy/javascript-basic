# man 帮助文档

```bash
man
  -f # 查看级别
  -k == apropos # 查看相关命令
```

* 查看命令的帮助  
* 查看可被内核调用的函数的帮助  
* 查看函数和函数库的帮助  
* 查看特殊文件的帮助(/dev)  
* 查看配置文件的帮助  
* 查看游戏的帮助  
* 查看其他杂项的帮助  
* 查看系统管理员可用命令的帮助  
* 查看和内核相关文件的帮助  

1. 选项帮助`ls --help`
2. shell 帮助`whereis cd`，`help cd`
3. info 命令  
`-回车：进入子帮助页面（带有*号标记）`  
`-u 进入上层页面`  
`-n 进入下一个帮助小结`  
`-p 进入上一个帮助页面`  
`-q 退出`
